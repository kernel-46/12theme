Pratyaya is a real-time voice-to-voice assistive layer for Karnataka's 1092 helpline. It sits between the citizen and the agent, ensuring every issue is accurately heard, interpreted, and verified — before any response is committed. It never replaces the human, never auto-resolves a call, and never acts without confirmation.

🎙️

Multilingual streaming ASR

Whisper Large v3 stands in for AI4Bharat IndicConformer + BharatGen Shrutam2 — handles native Kannada-English / Hindi-English code-mixing with per-segment confidence.

🗣️

Dialect-aware routing

Lexical heuristics + LLM classification distinguish Bangalore-standard, Dharwad, Mangaluru and Hyderabad-Karnataka Kannada. Verification paraphrases use the citizen's own dialect register.

🧠

India-native NLU

Sarvam-1 style structured extraction: issue type, urgency, location, named persons, factual claims, and an explicit clarification flag. Refuses to invent facts.

🚦

Confidence-gated state machine

VERIFIED · CLARIFY · HANDOVER. Disagreement between models, repeated clarifies, or extreme distress all formally escalate to the human agent.

💓

Six-dimensional sentiment

distress · urgency · anger · fear · confusion · calm — fused from lexical and prosodic signals, updated every turn with a live trajectory chart.

🤝

Graceful handover protocol

A short bridging line spoken in the citizen's dialect, dashboard pre-populated with full context — the citizen never has to repeat themselves.

🔐

PII redaction at the edge

Phone, Aadhaar, email, long digit blocks redacted before any data is logged or sent to a language model.

⛓️

Hash-chained audit ledger

Every turn, correction, confirmation, and handover is appended to a SHA-256 hash chain. Tampering with any past row breaks verification.

🌫️

Shadow-mode rollout

Per-call shadow toggle: the system listens, interprets, and updates the dashboard, but does NOT speak — until verification accuracy crosses the threshold per dialect.

📈

Civic sensor analytics

The same loop that prevents misunderstanding produces a real-time map of what citizens are calling about, from which districts, in which dialects, and with what emotional urgency.

🌐

Multilingual call summary

One tap to translate the latest interpretation into Kannada / Hindi / English — agents can hand off to a colleague in any language without re-explaining.

⚡

Distress alerts & shortcuts

Agent UI flashes and beeps when distress crosses 0.75. Keyboard shortcuts: H hand over, E export, V verify chain, Esc end call, Shift+T toggle theme.
